RhythmMeister

Typography on the web is difficult. It is hard to get different blocks in different columns to align in the same grid. Print has solved typography problems ages ago. Things like the baseline and kerning are industry standards, the web however evolving from hypertext does not use these standards.

The main problem lies in the use of line-height instead of a base-line. RhythmMeister provides a css extension in the form of a PostCSS plugin so you can use base-line instead of line-height. It provides this via a clever trick. It sets the line-height to the selected rows and than it adds a little padding to the top and the bottom.
