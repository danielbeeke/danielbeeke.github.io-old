Onitama

Onitama, a small but interesting game. It is a really nice board game, relatively new. It looks a bit like chess but the moves are determined by cards that have sets on them.

This is the first game I programmed. I learned about finite state machines and made this game in ES6, the new version of javascript.

Hobby project
2018
WebRTC, ES6
View on github!

My digital clone starts with a screen where you enter a room name so you can join a game. After choosing one, you can click on the link, share it with your friend and play the game.

Technically the user is connecting to websockets with the room name. When there are two players in a websocket's room the Nodejs server will connect them to each other via WebRTC. After that the websockets server is disconnected from the players.

While creating this game I learned a lot about state changes, ES6 and how game development is different from plain application building.
