Onitama

Onitama, a small but interesting game. It looks a bit like chess but the moves are determined by cards that have sets on them.

Hobby project
2018
WebRTC, ES6
View online!

It is a really nice board game, relatively new. My digital clone starts with a screen where you enter a key that will be used with websockets. After choosing one, you can click on the link, share it with your friend and play the game.

After connecting via websockets the players will be connected to only each other via WebRTC.

While creating this game I learned a lot about state changes, ES6 and how game development is different from plain application building.
