Haringvliet

A dozen of organisations from the Netherlands have bundled forces to advance and develop a region. This region has a lot of beautiful nature.

Our task was to create a news / blog website with an interactive map.

We did this in three iterations.

  1. We built the website with news, blogs and some project information.
  2. Then we built a first version of the map, with a map drawn by an illustrator.
  3. Then we made the map as shown in this showcase.
With Fonkel & Graphius, for WWF

The project was built with Drupal, Leaflet, Mapbox Studio Classic. I created the map layer with CartoCSS and created the initial architecture for the various components of the map.

This project was mostly made by me, for me that was really nice, building such a nicely designed project! It was designed by Graphius.

We had a different UI for mobile using our open sourced library 'Sprinkhaan'. These three screens show the use of Sprinkhaan. It mimics the user experience of other map platforms. It was built using the new web animation API. More recently I converted Sprinkhaan to vanilla ES6 code.

Bike routes can be uploaded via .gpx files. You can switch of all kinds of layers. We have a specific layer for the road labels and also one for the bike signs.

Things I have learned from this project are: a little bit of Qgis and a lot of CartoCSS. We build this while I was learning ES6. In this time I found a way to combine Drupal behaviours with transpiled ES6 code. I learned a lot of ES6 code and tricks while working on this project.
